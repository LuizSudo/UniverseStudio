# Universe Studio
A 3D Gravity simulation software written in C++

> [!IMPORTANT]
> This projects is in early development stage

## Build
1. Install the Dependencies
2. cd into `universtudio`
3. enter `make run`

## Dependencies
- OpenGL
- GLFW
- GLM
- Assimp
